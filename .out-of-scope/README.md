# Out Of Scope

Record durable "we are not doing this" decisions here so future agents do not repeatedly suggest the same rejected work.

Create one markdown file per rejected idea:

```text
.out-of-scope/<slug>.md
```

Suggested format:

```markdown
# <Rejected Idea>

## Decision

<wontfix / not now / outside product scope>

## Rationale

<why this is out of scope>

## Revisit When

<conditions that would make this worth reconsidering, or N/A>
```
